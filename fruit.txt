git init - This command is used to turn a folder into a git repo
git status - Used to check the status of the work we've done inside of the folder
git add <filename> - Used to stage work from the work area to the staging area
git commit -m "Message" - Used to create history of our work
git log - Used to check the list all of our commit


git branch - this allows us to create new branches in our repository
git branch <branch_name> - This allows us to create new branches in our repository 
git switch <branch_name> - This commans allows us to swicth from one branch to another 
git merge <name of the branch you wish to merge with the main branch>
